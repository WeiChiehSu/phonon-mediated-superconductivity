$$
由於計算\lambda _{q}和\alpha ^{2}F(\omega )需要在Fermi-surface進行雙重\delta積分,所以需要使用broadening(展寬)方法進行近似:
$$

$$
broadening:I=\int_{BZ}^{}dq \int_{BZ}^{}d\epsilon f(k,q)\delta (\epsilon _{n,K}-\epsilon _{F})\delta (\epsilon _{n^{'} ,k^{'}}-\epsilon _{F})
$$

$$
使用broadening時,在網格切割成的區間中用一組'類似,模糊'的光滑連率函數去處理出現階躍函數的地方(\sigma=degauss),將broadening引入後,積分近似為:
$$

$$
I\simeq \frac{\Omega _{BZ}^{2} }{N_{K} N_{q}} \sum_{k}^{} \sum_{q}^{}  \frac{1}{\sqrt{2\pi \sigma } }e^{-\frac{(\epsilon _{k}-\epsilon _{F})^{2}   }{\sigma ^{2} } } \frac{1}{\sqrt{2\pi \sigma } }e^{-\frac{(\epsilon _{k^{'}}-\epsilon _{F})^{2}   }{\sigma ^{2} } }   
$$

$$
\alpha ^{2}F(\omega )\simeq \frac{1 }{2N_{K} N_{q}N(\varepsilon _{F})\omega _{q}} \sum_{nn^{'}}^{} \sum_{k}^{} \sum_{q}^{}\left |  g_{nn^{'}}(k,q) ^{2} \right | \delta (\omega -\omega _{q})  \frac{1}{2\pi \sigma  }e^{-\frac{(\epsilon _{k}-\epsilon _{F})^{2}   }{\sigma ^{2} } } e^{-\frac{(\epsilon _{k^{'}}-\epsilon _{F})^{2}   }{\sigma ^{2} } }
